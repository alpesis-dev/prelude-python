This repo is to generate a project template by bash script.


Creating a project
------------------------

::

    $ wget https://raw.githubusercontent.com/KellyChan/project-template/master/create_project.sh
    $ ./create_project.sh

Project Template
------------------------

::

    ./project
        |---- venvs/
        |---- project/
                 |---- README.rst
                 |---- AUTHORS.rst
                 |---- LICENSE.rst
                 |---- RELEASES.rst
                 |---- docs/
                 |---- requirements/
